Section 2: Interactive Answers & Keys Config

Calculation workflow

Base path: equation given in the prompt (substitute) or recall from equation sheet. Optional steps (each +1 mark when ticked): rearrangement, unit conversion, significant figures.

Equation select step

Student chooses the correct equation from the sheet (or a dropdown). Not marked — informational only.

Substitution step

Structured mode shows one input per quantity in the equation. Free text uses a single line (legacy).

Unit conversion step

Example: question gives 1.2 km — step label “Convert 1.2 km to metres”, correct converted value 1200, tolerance 0.001 (student enters 1200 ± 0.001).

Rearrangement step

Student picks the correctly rearranged formula (or numeric rearrangement with values). Can be added to either equation-given or equation-recall questions.

Significant figures step

Final answer must be given to this number of significant figures (checked on the calculate input).

Final answer step

Correct value and tolerance are set in the answer key fields above. Optional message if the final number is wrong.

Wrong answer walkthrough (optional)

Shown step-by-step when the final answer is incorrect. Add one row per teaching step.

Optional custom feedback for each step is set under Feedback if wrong in the step panels above. Built-in defaults are used if left blank.

Note: If you leave the "Section 3: Step-by-Step Mark Checkpoints" completely blank below, this question will grade automatically using basic Required/Optional keyword algorithms.

Section 3: Step-by-Step Mark Checkpoints

Short Answer Questions: Each matched row awards 1 mark (AO-specific). If left blank, grading uses the required/optional keywords in Section 2 instead.
Numeric Questions: Feedback is configured in Section 2 (calculation workflow panels) — this section is not used.
MCQ: Add one generic remediation row (feedback field only). Shown below any per-option feedback from Section 2 when a student picks a wrong answer.
